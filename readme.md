・概要

冷笑系の居酒屋で働くゲーム
プレイヤーは店員となり、客の会話に首を突っ込んだり客と会話中に冷笑することで、酒の温度を下げられる 0℃を下回ると凍る 氷結とかがいいと思う
この店はキンキンに冷えた酒がウリなので、配膳と同時に冷笑して酒を上手く冷やしてから提供する

稀に熱血系の客が来店（松岡修造とか）
酒の温度がみるみる上がる強敵だが、高スコア

客の性格や雰囲気を読み取って選択肢で冷笑ワードを選んでいく
通常の客は十分お酒が冷えると酒を受け取るが、時間がかかりすぎたりお酒が熱くなりすぎると帰ってしまう
熱血系の客が来たらピンチだが、同時にチャンスでもある
「うおｗ」ボタンを連打して客の熱気とバトルし、提供の可否を決める
熱血系の客に提供に成功するとスコアはかなり高いものの、失敗すると熱気に押されて数秒間行動不能になる

提供に成功するとスコアと残り時間ボーナス獲得
時間切れ時のスコアでランキングを表示したいが、Github Pagesで実現できる？

・技術選定
Node.js + TypeScript 
Github Pagesでホストする
